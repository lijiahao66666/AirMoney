软件概述
1.2
哎呀钱 （AirMoney）是⼀款结合了前沿⼈⼯智能技术的现代化智能记账软件。它基于 Flutter 跨平台框架开
发，⽀持 Android、iOS、Web 及桌⾯端运⾏。软件不仅具备传统记账功能，更深度集成了腾讯混元⼤模型，

第⼋章 性能优化与安全设计
性能优化
8.1
1. 列表懒加载：最近记录列表采⽤ ListView.builder 实现懒加载，仅渲染可⻅区域
2. 数据缓存：积分余额同时存储于内存和 SharedPreferences，减少⽹络请求
3. 异步计算：数据库查询和 AI 请求均在异步线程执⾏，确保 UI 流畅
4. 流式响应：AI 对话采⽤ SSE 流式传输，实现打字机效果，提升⽤⼾体验
安全与隐私
8.2
1. 数据沙箱：应⽤遵循移动端沙箱机制，所有数据仅限应⽤内部访问
2. Token 安全：⽤⼾认证 Token 存储于 SharedPreferences，传输全程 HTTPS 加密
3. 本地优先：账单数据存储在本地 SQLite，不上传云端，保护⽤⼾隐私
4. 积分同步：仅积分余额通过加密通道与服务器同步，确保账⼾安全

1. 便捷记账：简洁的记账界⾯，⽀持⽀出/收⼊分类、多种⽀付⽅式
2. 智能分析：AI 驱动的消费分析，帮助⽤⼾反思消费⾏为
3. 买前咨询：购买前的理性决策助⼿，避免冲动消费
